What is the Document Object Model (DOM)?

The DOM (Document Object Model) is like a tree structure that shows how all the parts of a web page, such as its content and elements, are organized and connected.

How do you select an element from the DOM?

I can select DOM elements by using ID (getElementById), class (getElementsByClassName) and tag name (getElementsByTagName). For CSS selectors, I use querySelector for a single match or querySelectorAll for all matches. I also can access the DOM using properties like parentNode, children, nextSibling, and closest.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation means adding one event listener to a parent instead of many to kids. It helps catch events from child elements as they bubble up. This makes the code faster and easier to manage

How do you manipulate an element's attributes and styles using the DOM?

We can change an element's attributes using getAttribute, setAttribute, and removeAttribute. For styles, use the style property to set, get, or remove CSS styles directly.
