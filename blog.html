What is the DOM in JavaScript?

The DOM (Document Object Model) is an interface provided by web browsers that allows programs and scripts, such as JavaScript, to dynamically access and update the content, structure, and style of a web document (usually an HTML or XML document). It represents the document as a tree of nodes, where each node corresponds to an element, attribute, or text in the HTML document

How can you select an element from the DOM using JavaScript?

You can select elements from the DOM using various methods such as: document.getElementById(id) – selects an element by its id. document.getElementsByClassName(className) – selects all elements with a specific class. document.getElementsByTagName(tagName) – selects all elements with a specific tag name. document.querySelector(selector) – selects the first element that matches a CSS selector. document.querySelectorAll(selector) – selects all elements that match a CSS selector.

How do you create a new element and add it to the DOM?

You can create a new element using document.createElement() and append it to the DOM using methods like appendChild() or insertBefore().

How can you modify the content or attributes of a DOM element?

You can modify the content of an element by changing its innerHTML, textContent, or innerText properties. You can modify attributes using the setAttribute() or directly setting the attribute property.

What is event delegation in the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element rather than adding multiple event listeners to child elements. The event listener takes advantage of event bubbling, allowing it to listen for events triggered by child elements. This is useful for dynamically added elements, improves performance, and reduces memory usage
